=====
Lists
=====

.. default-domain:: mongodb

You can create *ordered* or *unordered* lists in {+rst+}.

Ordered Lists
-------------

Ordered lists show a number for each item in the list. Create an ordered list by
starting a line with ``#.``, or by using a number followed by a ``.``. Using
``#.`` in place of numbers prompts Snooty to generate list numbers automatically.

Nested Lists
------------

You can create nested lists with both ordered and unordered lists. Create a
nested list by first creating a list, then creating a second list separated by
a blank line and indented. Indent the second list to the same level of
the text of the parent list.
